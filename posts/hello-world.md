# Hello, world

This is the first post. Every file in the `posts/` folder becomes a page.

## How to add a post

1. Drop a `.md` file into `posts/`.
2. Add frontmatter at the top:

```md
---
title: "My Post Title"
date: "2026-07-02"
description: "Short summary shown in the list."
---
```

3. Write markdown below. That's it.

The **filename** (minus `.md`) becomes the URL slug, so this file lives at
`/posts/hello-world`. Posts are sorted newest-first by `date`.

> Standard markdown works: lists, code blocks, tables, quotes, links.
